Info

This website is intended to be an online guitar effects processor. To use this site, connect a guitar to your computer and select Use Guitar. There is also the Use Sample option. This allows you to experiment with the different effects using a prerecorded sample. I recommend using a Mac with an audio interface while using the Chrome browser for best performance.
